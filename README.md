Welcome to Science Around the Board! This is a pedagogical game intended to incentivise learning of any scientific contect through fun and engaging dynamics. 

Players choose from a range of topics previously set up by their instructor and roll a dice around the board to conquer tiles by answering questions.


**Instructor guidelines to set up the questions**

1) **Required columns** - do NOT rename.
The Questions sheet must contain these columns exactly:
id
question
option1
option2
option3
option4
correctIndex
explanation
bigTopic
module
theme
subtheme
type

2) **What each column means**:
id            Unique question ID (e.g., q001, m010, pre003)
question      The question prompt
option1–4     Multiple-choice answers (leave blank if unused)
correctIndex  Number 1–4 for correct option (still required for surveys)
explanation   Shown only for milestone questions
bigTopic      Topic shown on main menu (e.g., 16S, Shotgun, Both)
module        Sub-topic (e.g., Overview, Advanced). Can be blank.
theme         Board category (e.g., Sample Prep, Sequencing)
subtheme      Sub-category of theme (e.g., Extraction, Library Prep)
type          One of: property, core, milestone, pre, post
